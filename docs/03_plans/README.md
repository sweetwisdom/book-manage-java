# 开发计划目录

本目录用于保存复杂任务、重构任务、多步骤任务或用户明确要求保存的开发计划。

## 使用场景

| 场景 | 是否保存 |
| --- | --- |
| 新增完整业务模块 | 保存 |
| 数据库结构变更加接口开发 | 保存 |
| 跨多个类或多层的重构 | 保存 |
| 需要多轮对话持续推进的任务 | 保存 |
| 单文件小改动或简单问题修复 | 不强制 |

## 命名规范

```text
YYYYMMDD_plan_任务名称_v版本号.md
```

示例：

```text
20260702_plan_借阅模块开发_v1.md
```

## 建议结构

- 背景
- 目标
- 范围
- 执行步骤
- 验证方式
- 风险和注意事项
