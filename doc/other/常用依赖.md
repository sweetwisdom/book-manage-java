像 RuoYi 这种典型单体后台项目，一般不只是你上面那几个依赖。

一个完整的后台管理系统，依赖大概会分成下面几层：

---

# 1. Web层

```xml
spring-boot-starter-web
spring-boot-starter-validation
```

负责：

* Controller
* 参数校验
* JSON转换

例如：

```java
@RestController
@RequestMapping("/system/user")
```

---

# 2. 数据库层

## MyBatis Plus

```xml
mybatis-plus-boot-starter
```

或者：

```xml
mybatis-spring-boot-starter
```

---

## MySQL驱动

Spring Boot 3：

```xml
<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
</dependency>
```

---

## 数据库连接池

现在默认：

```text
HikariCP
```

已经包含在：

```xml
spring-boot-starter-jdbc
```

里。

---

# 3. 安全认证

RuoYi 最核心的部分之一。

## Spring Security

```xml
spring-boot-starter-security
```

提供：

```java
@PreAuthorize
UserDetailsService
AuthenticationManager
```

---

## JWT

常见：

```xml
jjwt-api
jjwt-impl
jjwt-jackson
```

或者：

```xml
java-jwt
```

用于：

```text
登录
Token
权限认证
```

---

# 4. Redis

缓存和登录态。

```xml
spring-boot-starter-data-redis
```

用于：

```text
验证码
Token缓存
权限缓存
分布式锁
```

---

# 5. AOP

```xml
spring-boot-starter-aop
```

RuoYi大量使用。

例如：

```java
@Log(title="用户管理")
```

底层：

```java
@Aspect
```

记录操作日志。

# 14. 单体项目最常见组合

你以后看大部分后台项目，基本就是：

```xml
spring-boot-starter-web
spring-boot-starter-validation

mybatis-plus-boot-starter
mysql-connector-j

spring-boot-starter-security
jjwt

spring-boot-starter-data-redis

spring-boot-starter-aop

springdoc-openapi-starter-webmvc-ui

easyexcel

hutool-all

spring-boot-starter-test
```

可以理解成：

```text
Web          -> 接口
MyBatisPlus  -> 数据库
MySQL        -> 存储
Security     -> 权限
JWT          -> 登录
Redis        -> 缓存
AOP          -> 日志
Swagger      -> 接口文档
EasyExcel    -> 导入导出
Hutool       -> 工具类
Test         -> 单元测试
```

如果你准备深入看 RuoYi、JeecgBoot、SpringBoot Admin 这类单体项目，建议优先掌握这几个模块的学习顺序：

```text
1. Spring MVC
2. MyBatis Plus
3. Spring Security + JWT
4. Redis
5. AOP
6. Quartz
7. Swagger/OpenAPI
```

这 7 个模块基本覆盖了 80% 的企业级单体 Java 后台开发。
