# Fat Jar 打包与部署指南

## 前置条件

- JDK 11+
- Maven 3.6+
- MySQL 5.7+ / 8.0+

## 一、打包

```bash
# 进入项目根目录
cd book-manage

# 清理并打包（跳过测试，加速打包）
mvn clean package -DskipTests

# 打包产物位置
target/book-manage-0.0.1-SNAPSHOT.jar
```

## 二、本地启动

### 基础启动

```bash
java -jar target/book-manage-0.0.1-SNAPSHOT.jar
```

应用启动后访问: http://localhost:8080

### 指定数据库密码

```bash
# Linux / macOS
export DB_PASSWORD=your_password
java -jar target/book-manage-0.0.1-SNAPSHOT.jar

# Windows CMD
set DB_PASSWORD=your_password
java -jar target/book-manage-0.0.1-SNAPSHOT.jar

# Windows PowerShell
$env:DB_PASSWORD="your_password"
java -jar target/book-manage-0.0.1-SNAPSHOT.jar
```

### 修改端口

```bash
java -jar target/book-manage-0.0.1-SNAPSHOT.jar --server.port=9090
```

### 指定外部配置文件

```bash
java -jar target/book-manage-0.0.1-SNAPSHOT.jar --spring.config.location=file:./application.yml
```

## 三、服务器部署

### 1. 上传 jar 到服务器

```bash
scp target/book-manage-0.0.1-SNAPSHOT.jar user@server:/opt/book-manage/
```

### 2. 后台运行

```bash
nohup java -jar /opt/book-manage/book-manage-0.0.1-SNAPSHOT.jar \
  --spring.profiles.active=prod \
  > /opt/book-manage/app.log 2>&1 &
```

### 3. Systemd 服务（推荐）

创建 `/etc/systemd/system/book-manage.service`:

```ini
[Unit]
Description=Book Manage Application
After=syslog.target network.target mysql.service

[Service]
User=app
WorkingDirectory=/opt/book-manage
Environment="DB_PASSWORD=your_secure_password"
ExecStart=/usr/bin/java -jar /opt/book-manage/book-manage-0.0.1-SNAPSHOT.jar
SuccessExitStatus=143
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动服务:

```bash
sudo systemctl daemon-reload
sudo systemctl enable book-manage
sudo systemctl start book-manage
sudo systemctl status book-manage
```

### 4. 查看日志

```bash
# 实时查看日志
tail -f /opt/book-manage/app.log

# 或通过 systemd
journalctl -u book-manage -f
```

## 四、Docker 部署

### Dockerfile

```dockerfile
FROM openjdk:11-jre-slim
WORKDIR /app
COPY target/book-manage-0.0.1-SNAPSHOT.jar app.jar
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]
```

### 构建与运行

```bash
# 构建镜像
docker build -t book-manage .

# 运行容器
docker run -d \
  -p 8080:8080 \
  -e DB_PASSWORD=your_password \
  -e SPRING_DATASOURCE_URL=jdbc:mysql://host.docker.internal:3306/test \
  --name book-manage \
  book-manage
```

## 五、常用运维命令

```bash
# 查看进程
ps aux | grep book-manage

# 停止应用（优雅关闭）
kill -15 <PID>

# 强制停止
kill -9 <PID>

# 检查端口占用
netstat -tlnp | grep 8080
```

## 六、配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `server.port` | 8080 | 应用端口 |
| `DB_PASSWORD` | 123456 | 数据库密码（环境变量） |
| `spring.profiles.active` | default | 激活的配置文件 |

## 七、健康检查

```bash
# 应用信息
curl http://localhost:8080/actuator/info

# 健康状态（需添加 spring-boot-starter-actuator 依赖）
curl http://localhost:8080/actuator/health
```
